# Curty Capital — CRM Panel

Panel completo de CRM para gestão de clientes, pipeline comercial e publicidade.

## Funcionalidades

- **Clientes**: Gerenciamento de clientes ativos, inativos e pausados
- **Pipeline**: Funil de vendas com etapas (Lead → Diagnóstico → Proposta → Fechado)
- **Publicidade**: Rastreamento de campanhas Facebook e Google Ads
- **Forecast**: Cálculo automático de forecast 30/60/90 dias
- **Relatórios**: Client report e agency report

## Stack

- Frontend: HTML5 + CSS3 + JavaScript
- Backend: Supabase (PostgreSQL)
- Deployment: Vercel

## Como usar

1. Abra `index.html` no navegador
2. Configure as variáveis de ambiente do Supabase
3. Todos os dados serão sincronizados com o banco de dados

## Deploy

```bash
bash deploy.sh
```
